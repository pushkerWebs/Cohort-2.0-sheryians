# 🚀 Sheryians Cohort 2.0 Landing Page Clone

This project is a **responsive frontend clone** of the Sheryians Coding School Cohort 2.0 landing page.

🔗 Live Website:
https://pushkerwebs.github.io/Cohort-2.0-sheryians/

---

## 📌 Project Overview

This project was built as part of my **learning journey**, where I practiced creating modern UI layouts and improving responsiveness.

The goal was to **replicate a real-world landing page** and understand how production-level websites are structured.

---

## 🛠️ Tech Stack

* HTML5
* SCSS (Sass)
* CSS3
* JavaScript

---

## ✨ Features

* 📱 Fully responsive design
* 🍔 Mobile hamburger menu with full-screen overlay
* 🎨 Clean UI inspired by modern design
* ⚡ Smooth scrolling effect
* 💰 Price section with strike-through styling
* 📦 Structured layout using Flexbox & Grid

---

## 📱 Responsiveness

The website is designed to adapt across different screen sizes.

> ⚠️ Best experience on **larger screens (desktop/laptop)**
> Mobile responsiveness is implemented and continuously being improved.
